# Primitives

## **6 Types of Primitive Data Types:**

1. `undefined`
2. `null`
3. `number`
      - NaN is number type
4. `string`
5. `boolean`
6. `symbol` (introduced in ES6)

## **Falsey Values:** 

- `false`
- `0` also `-0`
- `null`
- `undefined`
- `''`, `""`, ``... empty strings
- `NaN` 


## **Non-primitive Data types:**

1. Object
    - anything not primitive is Object type
    - array is type Object
    - function is type Object
